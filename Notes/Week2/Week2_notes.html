Pretty Code

Think of the code as in boxes. Everytime you create a new tag you want to tab it in. Elements at the same level should be indented at the same amount.

Rule of thumb is that the indent should be 2 spaces. The reason for that is because elements will be nested and you want to keep code at a level at which you are able to notice a difference

Tags and attributes should all be lowercase.Attibute values should always be enclosed in quotes.

CSS: Class vs ID

Classes

You want to use class attributes to categorize elements so that you can style all of the elements within that class.

ID

You want to use IDs as unique identifiers, you can style elements by id.
