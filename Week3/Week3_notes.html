CSS Review

Some common css properties:

  • font-family
  • color
  • background-color
  • width
  • height
  • padding

These properties should go in the your style tags at the top of your html file, in-line within the specific element you wish to style, or in a separate style sheet that you would reference using the link tag. The style sheets should contain elements along with brackets containing the properties you wish to assign to said elements.

Condense properties by referencing multiple elements so that your code is not too long or redundant. For example: "h1 h2 h3" references three header elements so all properties inside those brakets will affect said elements.
